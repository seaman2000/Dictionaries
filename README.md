# 🐍 Python Fundamentals – Dictionaries

This section contains tasks and exercises focused on **Dictionaries** in Python.

Dictionaries are one of the most powerful built-in data structures. They store data in **key-value pairs**, allowing fast access, modification, and organization of information.

## Topics Covered

## Topics Covered

### Lab
The **Lab** tasks focus on building a solid foundation in working with dictionaries.

They introduce:
- Creating and modifying dictionaries
- Accessing elements by key
- Iterating through keys and values
- Basic dictionary operations

These tasks are designed to help you understand the core concepts in a simple and practical way.

---

### Exercise
The **Exercise** section contains more advanced problems that require combining dictionaries with logical thinking.

You will practice:
- Counting and aggregating data
- Working with nested dictionaries
- Sorting dictionaries by key or value
- Processing and organizing information

These tasks simulate real-world scenarios and improve problem-solving skills.

---

### Bonus Exercises
The **Bonus Exercises** are optional but more challenging.

They focus on:
- Complex data structures
- Edge case handling
- Writing optimized and clean code
- Combining multiple programming concepts
